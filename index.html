Vehicle Detection and Tracking Algorithm

This project provides a solution to reduce false detection under high-density traffic conditions. Computer vision is used to detect vehicles on the road and extract the trajectory.

BikeSim

BikeSim is a tool to simulate the mixed traffic flow in Asia. It can generate the traffic flow based on different theorem and display the animation. Using historical data is also available.

Receipt Lottery

For those who are not familiar with phones, likes elders, this program can capture the image from webcam and recognize the numbers of receipt. The result will be stored as txt files so that user can easily search the numbers.

GUI for behavior analysis

This project provides a graphic user interface to assist user in analyzing the behavior of motor riders. GUI shows both the video and sensor data such as acceleration. It can be used to figure out if there's any relationship between specific behaviors and data patterns.
